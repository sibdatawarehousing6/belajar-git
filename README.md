## Class

Pada github kali ini terdiri 2 class di python:
1. Warga
2. Satpam

### Warga

Memiliki NIK dan dapat melakukan voting

### Satpam

Memiliki sifat yang serupa dengan warga dan memiliki fitur tambahan:
1. Menjaga TPU
2. Istrahat ketika bekerja
3. Lembur untuk bekerja
